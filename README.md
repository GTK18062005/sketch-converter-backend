# 🎨 Sketch Converter Backend

A powerful Spring Boot application that converts images into artistic sketches using OpenCV. This backend provides REST APIs for image processing with multiple artistic styles.

## ✨ Features

- 🖼️ **Multiple Artistic Styles**
  - ✏️ Pencil Sketch
  - 🎨 Watercolor Effect
  - 🖼️ Oil Painting
  - ⚫ Charcoal Drawing
  - 🎨 Artistic Sketch
  - 📜 Vintage/Sepia

- 🎛️ **Custom Parameters**
  - Adjustable blur size for smoothness control
  - Adjustable contrast for darkness/lightness control

- 🚀 **High Performance**
  - OpenCV-powered image processing
  - Efficient memory management
  - Fast conversion times

- 🌐 **RESTful API**
  - Easy integration with frontend applications
  - CORS enabled for cross-origin requests
  - JSON response format

## 🛠️ Technology Stack

- **Java 17** - Core programming language
- **Spring Boot 3.2.0** - Application framework
- **OpenCV 4.12.0** - Image processing library
- **Maven** - Build automation tool
- **Docker** - Containerization

## 📋 Prerequisites

### For Local Development
- Java 17 or higher
- Maven 3.6+
- OpenCV installed locally (for Windows/Mac)
- Git

### For Docker Deployment
- Docker Desktop
- Docker Compose (optional)
